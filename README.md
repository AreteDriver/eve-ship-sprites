# EVE Link

EVE Online ship sprite generation tools.

## Scripts

- `render_ship.py` - Render individual ship sprites
- `generate_sprites.sh` - Batch sprite generation
- `generate_spritesheets.sh` - Create sprite sheets

## Usage

```bash
python render_ship.py <ship_id>
./generate_sprites.sh
```
